# Handwritten Notes — Transcriptions

This folder contains AI-generated transcriptions of handwritten clinical notes from your GP record, alongside the original scanned images they were transcribed from.

## How to use this folder

Each transcription file references the source image filenames it was produced from. The original scanned images are in this same folder so you can compare the transcription against the handwriting at any time.

## What is included

- **Transcription files** (.md) — Each contains a table with three columns: Date, Transcription (faithful to the original handwriting), and Layman Interpretation (plain-English explanation of medical terms and abbreviations)
- **Original scanned images** — The source images from your GP record, exactly as received in the Subject Access Request response

## What is excluded

- **Dosages** — Medication names are transcribed but dosage figures are deliberately excluded. Prescription data including dosages is available in structured form elsewhere in your GP record and is far more reliable than attempting to read handwritten figures. If you need a specific dosage from an older record, refer to the original scanned image

## Accuracy

These transcriptions were produced using AI on a reasonable-endeavours basis. AI handles most handwriting well, but doctor's handwriting varies considerably and some words may be misread or marked as illegible. Where the AI is uncertain about a word, it is marked with `[?word?]` notation. Completely unreadable words are marked `[illegible]`.

**Important:** This is data organisation, not medical advice. Always consult a qualified healthcare professional before acting on any information in your health archive. Verify any critical information against the original scanned images in this folder.
